Fundamentals of CSS
_____________________
> Primo metodo (inline)
```
<body style="background-color: blue"></body>
```

> Secondo metodo (internal)
```
<head>
    <style>
      body {
        background-color: blue;
      }
    </style>
 </head>
```

> Terzo metodo (external)
```
<head>
  <link rel="stylesheet" type="text/css" href="styles.css">
</head>
```
